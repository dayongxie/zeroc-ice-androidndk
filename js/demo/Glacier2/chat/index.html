Chat Demo Readme

This example demonstrates the use of a Glacier2 session to create a very simple chat client.

First follow the instructions from the C++ Glacier2 chat demo README to start the server.

  • To use the C++ server you'll need a C++ compiler compatible with your Ice for JavaScript distribution.

Once the chat server is ready you can sign in using any username and password.

The client is configured to use WSS secure endpoints when the page is loaded over HTTPS and WS unsecure endpoints when loaded over HTTP.
